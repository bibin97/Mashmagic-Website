# Mashmagic Website

A premium, modern educational platform for **Mash Magic**. This repository contains both the frontend and backend components of the website.

## 🚀 Features

- **Dynamic Animations**: Powered by GSAP and Framer Motion for a smooth, high-fidelity user experience.
- **Sleek UI**: Modern glassmorphism design built with React and Tailwind CSS.
- **Responsive**: Fully optimized for all device sizes.
- **Optimized Video Hosting**: Integration with Cloudinary for efficient video playback and faster load times.

## 📁 Project Structure

```text
.
├── Frontend/          # React + Vite application
├── backend/           # Node.js + Express backend
└── vercel.json        # Vercel deployment configuration
```

## 🛠️ Tech Stack

- **Frontend**: React, Vite, Tailwind CSS, GSAP, Framer Motion, Lenis Scroll.
- **Backend**: Node.js, Express, MongoDB.
- **Deployment**: Vercel (Frontend), [Backend Deployment Link if any].

## 💻 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/bibin97/Mashmagic-Website.git
   ```

2. **Frontend Setup**:
   ```bash
   cd Frontend
   npm install
   npm run dev
   ```

3. **Backend Setup**:
   ```bash
   cd backend
   npm install
   # Create a .env file and add your credentials
   npm start
   ```

## 🌐 Deployment

The frontend is configured for automatic deployment on **Vercel**. The root `vercel.json` ensures that the build process correctly targets the `Frontend` directory.
